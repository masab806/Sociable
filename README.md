# 💬 Real-Time Chat App (React Native + Socket.IO)

A full-stack real-time chat application built with **React Native (Expo)**, **Node.js**, **Express**, **Socket.IO**, and **Drizzle ORM**.

Supports:
- 🔐 Authentication
- 💬 Real-time messaging
- 🧵 Conversations
- ⚡ Instant updates via WebSockets

---

## 🚀 Tech Stack

### Frontend (Mobile)
- React Native (Expo)
- TypeScript
- Expo Router
- Zustand (state management)
- Socket.IO Client

### Backend
- Node.js
- Express.js
- Socket.IO
- Drizzle ORM
- PostgreSQL (or your DB)
